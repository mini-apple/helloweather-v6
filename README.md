# 🌤 부산대학교 헬로웨더 예보게임 채점 서비스 - Hello Weather Forecast Judge

## 1. 서비스 간단 소개

- 홈페이지 링크:  https://helloweather-v6.firebaseapp.com/
- Hello Weather Forecast Judge는 헬로웨더에서 진행하는 예보게임의 채점을 도와주는 온라인 서비스입니다.
- 기존 채점표를 이용했던 채점 방식과 기준을 그대로 온라인 홈페이지에 적용하여 보다 쉽게 예보 게임 채점을 할 수 있도록 돕고 있습니다.
- 예보게임 방을 만들어서 멤버들과 동시에 채점을 할 수 있는 온라인 채점 방법과 각자 채점 할 수 있는 오프라인 채점방식이 있습니다. 기능과 채점 방법에 대해서는 아래에 따로 자세히 소개하겠습니다.
- 이 웹페이지는 헬로웨더 네이버 카페처럼 검색 엔진을 통해서는 검색되지 않도록 설정해두었습니다. 아래 주소를 직접 입력하거나 북마크를 설정해서 접속해주시면 됩니다.

## 2. 서비스 기능, 채점 방법

- 주로 노트북보단 스마트폰을 통해 이용할 것 같아서, 스마트폰 화면을 기준으로 설명하겠습니다.

## 2-1. 오프라인 채점 (비로그인 채점 방법)

### 가. 예보게임 인도자

**1. 홈화면 → 상단 메뉴바 클릭 → 정답 코드**
- 인도자는 자신이 기획한 예보게임의 정답 코드를 만들어서 공유해줘야합니다.

<img src="https://github.com/user-attachments/assets/b6cca01b-5f0f-4397-928d-451d88b5a52c" width="300px" height="570px">
  
**2. 정답코드 생성**
- 지역 순서는 예보게임 종이의 순서대로 위에서부터 지역1, 지역2 입니다.
- 답안 입력하기에 예보게임의 정답을 입력한 후 create 버튼을 눌러 정답코드를 생성합니다.
- 복사 아이콘을 누르면 클립보드에 복사됩니다. 생성된 정답 코드를 복사해서 채점자들에게 카카오톡 등을 통해 정답코드를 공유해주세요.

<img src="https://github.com/user-attachments/assets/30fa26e6-a300-4336-8acb-0af9b950d814" width="300px" height="570px">

### 나. 예보게임 채점자

**1. 홈화면 → 상단 메뉴바 클릭 → 채점하기**
- 채점자는 먼저 채점하기 메뉴에 들어갑니다.

<img src="https://github.com/user-attachments/assets/1001b2e4-b960-419c-828f-d6eefa25a885" width="300px" height="570px">

**2. 정답 코드 입력하기**
- 인도자가 공유해준 정답 코드를 정답 코드에 입력해주세요.
- 정답 코드를 입력하고, 반드시 옆에 엔터 버튼을 눌러줘야 정답 코드가 시스템에 올바르게 입력됩니다.
- 올바르게 입력되면 아래 화면과 같이 정답 코드의 내용을 확인할 수 있습니다.

<img src="https://github.com/user-attachments/assets/065bf5ab-00c6-4117-a97c-dc0e6290539e" width="300px" height="570px">

**3. 나의 답안 및 근거 개수 입력하기**
- 내가 예보게임 종이에 작성한 답안을 해당 란에 입력해줍니다.
- 예보게임 종이에 작성한 근거 개수만큼 근거 개수를 입력해줍니다. (좌우 스크롤로 조작)

<img src="https://github.com/user-attachments/assets/ea91a63f-dd07-4c7e-a638-a314ca32d6b2" width="300px" height="570px">

**4. 채점하기**
- Score 버튼을 누르면 입력한 정답코드와 나의 답안을 비교하여 채점표를 토대로 점수가 매겨집니다.

<img src="https://github.com/user-attachments/assets/9f8d4fee-efe0-49dd-8b46-de4588c89b4e" width="300px" height="570px">

## 2-2. 온라인 채점 (로그인 채점 방법)

### 가. 사전 작업 (로그인 / 회원가입)

- 온라인 채점은 모든 멤버가 로그인을 해줘야합니다.
- 카톡에서 공유해서 링크로 접속할 경우 카카오 브라우저의 보안 정책 이슈로 인해 구글 로그인이 아마 안될겁니다. 이메일 비밀번호로 로그인 할 수 있으나 귀찮기 때문에 **크롬 브라우저** 사용을 강력히 권장합니다.

### *회원가입

**1. 우측 메뉴 상단 아바타 클릭 → 로그인**

<img src="https://github.com/user-attachments/assets/49daa7c4-b375-43c8-a3df-710cf3bdf935" width="300px" height="570px">

**2. 아래로 스크롤 후 → 계정 생성하기**
- 아래로 스크롤 하면 계정 생성하기 링크가 있습니다. 이 링크를 클릭해주세요.

<img src="https://github.com/user-attachments/assets/a58581b9-c981-45ec-9bec-e94605f4d973" width="300px" height="570px">

**3. 구글 회원가입 / 이메일 회원가입**
- 자신이 편한 방법으로 회원가입을 하면 됩니다.
- 이름, 학번, 기수, 활동학기를 입력해주고 회원가입을 하면 됩니다.
- Space Name은 고유한 짧은 계정 식별자로 인스타그램 아이디 정도로 생각해주면 됩니다.
- 나중에 프로필 정보는 수정할 수 있습니다.
- 정보를 입력한 후 계정 생성하기 버튼을 눌러주면 계정이 생성됩니다.
- 이메일 회원가입도 동일합니다.

<img src="https://github.com/user-attachments/assets/15adab5d-7131-4956-9fa3-2d1b0bb7d796" width="300px" height="570px">

### *로그인

1. 우측 메뉴 상단 아바타 클릭 → 로그인 → 자신이 선택한 방법으로 로그인

### 나. 인도자의 예보 게임 생성

**1. 좌측 상단 메뉴 - 예보게임**

<img src="https://github.com/user-attachments/assets/96ac547b-566a-41d3-8a18-fc4a822f4fa6" width="300px" height="570px">

**2. 새 예보게임 등록하기**

<img src="https://github.com/user-attachments/assets/4a3060ca-c178-41b5-bf1a-4fc96846cfdb" width="300px" height="570px">

**3. 예보 게임 정보 및 정답입력하기**
- 예보 게임 일시와 예보 지역을 입력해주세요.
- 예보 게임이 크게 두 종류가 있습니다.
    1. 인도자가 이전 특정 날짜를 선택해서 예보 게임을 진행
    2. 예보 게임일의 다음날 15시를 기준으로 예보 게임 진행
- 1번의 경우 예보게임을 진행할 때 정답이 이미 있으므로 ‘미리 입력하기’를 선택해 정답을 시스템에 미리 입력해둡니다. (’채점할 때 입력하기’를 선택해도 상관은 없습니다.)
- 1번에서 정답을 입력해두어도 인도자를 제외한 다른 멤버들은 해당 예보게임이 채점되기 전까지 정답을 볼 수 없습니다.
- 2번의 경우 예보게임을 진행할 때 정답을 알 수 없으므로 ‘채점할 때 입력하기’를 선택합니다.
- 2번은 예보 게임의 정답이 나왔을 때 인도자가 정답을 입력해서 한 번에 멤버 전원을 채점 시킬 수 있습니다.

<img src="https://github.com/user-attachments/assets/f7d7466d-0710-43da-a8e5-6f2adb512a33" width="300px" height="570px">

<img src="https://github.com/user-attachments/assets/8d96aaac-74f2-4425-abd6-9501a33e6cfe" width="300px" height="570px">

- 예보게임 등록하기 버튼을 눌러서 입력한 정보를 바탕으로 예보 게임을 생성할 수 있습니다.

**4. 예보게임 생성 및 관리**

- 예보게임이 등록되면 아래 사진과 같이 목록이 뜹니다.
- ‘진행중’ 버튼을 눌러서 예보게임에 접속할 수 있습니다.

<img src="https://github.com/user-attachments/assets/c4bcc6a6-54e2-4035-b168-ba524ee37af6" width="300px" height="570px">

- ‘채점 관리 페이지로 가기’ 링크는 예보게임을 생성한 인도자에게만 보입니다.
- 해당 페이지로 이동하면 멤버들의 정답 입력 현황을 볼 수 있으며, 예보게임을 마감하고 일괄 채점을 진행할 수 있습니다. 멤버들의 정답 입력 과정을 설명하고 채점 과정을 설명하겠습니다.

<img src="https://github.com/user-attachments/assets/56f534ea-30a6-478b-953f-f603f793ae1b" width="300px" height="570px">

### 다. 참여자의 정답 입력

**1. 우측 상단 메뉴 아이콘 → 로그인 하기 → 메뉴 → 예보게임**

<img src="https://github.com/user-attachments/assets/30c87522-733d-4783-be08-e148b279a71a" width="300px" height="570px">

**2. 예보게임 목록 → 진행중 버튼 클릭**

<img src="https://github.com/user-attachments/assets/1a3a500b-2ddd-4a59-ac93-bac728c4d9d1" width="300px" height="570px">

**3. 정답 입력하기 → ‘입력 확인하기’ 버튼 클릭 → ‘정답 제출하기’ 버튼 클릭**
- 정답 입력하기에서 나의 정답을 입력해줍니다.
- 입력확인하기 버튼을 눌러 나의 정답이 시스템에 올바르게 입력되었는지 확인합니다.
- 정답 제출하기를 눌러서 정답을 서버에 전송합니다.
- 정답 제출하기를 누른 이후에도 인도자가 채점하기 전에는 다시 접속하면 정답을 수정할 수 있습니다.
- 정답 제출하기까지 하면 참여자의 할 일은 끝납니다.
